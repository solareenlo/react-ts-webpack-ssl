# react-ts-webpack-ssl
「Typescript + Express + React + Webpack でSSR(サーバサイドレンダリング)を検証」を試したみた記録です.  
きちんとは動きませんが忘備録です.

# Usage
```
git clone https://github.com/solareenlo/react-ts-webpack-ssl.git
cd react-ts-webpack-ssl
npm install
npm run build
npm run start
```
`localhost:3000`で見えると思います.
